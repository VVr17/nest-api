# Pet Support API

This is the backend REST API for the Pet Support web application, which allows users to browse and create notices related to pets. 
The API is built with Nest.js and uses MongoDB as its database.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
